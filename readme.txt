Allows you to use a Fastcase API key to add court opinions to your Wordpress platform.

== Description ==

THis plugin creates a custom post type called Opinions and requires an API key from Fastcase. Once you have the API key you can save it under settings. With the API key in place you can enter a valid citation for a case and retreive it from  Fastcase. The HTML of the court opinion is then used to create a new Opinion in Wordpress.
